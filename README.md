# saltan
Calculate salt anagrams
## Installation
1. Install nim
### Installing Saltan
1. Run `make release`
### Installing JSaltan
1. Install the Closure Compiler
2. Run `nimble install ajax`
3. Run `make jsrelease`
## Installing Both
1. Install the Closure Compiler
2. Run `nimble install ajax`
3. Run `make allrelease`
## Usage:
### Saltan
`saltan [word]`
### JSaltan
1. Run `python -m SimpleHTTPServer`
2. Point your browser to localhost:8000
